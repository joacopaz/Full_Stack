HTML Tables

The table structure in HTML is enclosed by a <table> tag. The table contains a header, a body and a footer.

Tag Use
<table> Opens a new table.
<thead> Opens a table header
<tr> Creates a new row.
<th> Adds data (column) to a row in the form of a header. A scope attribue can be added (scope="row") to the first th to declare it as a row header.
<tbody> Opens the table body.
<td> Table data, adds columns to already created rows. They nest inside of a <tr> tag.
<tfoot> Opens the table footer.

Example

<table> <thead> <tr> <th scope="row"> Row 1, Col 1 - Header <\th> <th> Row 1, Col 2 - Header <\th> <\tr> <\thead> <tbody> <tr> <td> Row 2, Col 1 <\td> <td> Row 2, Col 2 <\td> <\tr> <\tbody> <tfoot> <tr> <td> Row 3, Col 1 - Footer <\td> <td> Row 3, Col 2 - Footer <\td> <\tr> <\tfoot> </table>

Result

Row 1, Col 1 - Header Row 1, Col 1 - Header
Row 2, Col 1 Row 2, Col 2
Row 3, Col 1 - Footer Row 3, Col 2 - Footer
